# 🏃 九鼎 runners · 山海经童话 🤖

这是一个基于《九鼎 runners》创作的 2026 级 AI 童话交互门户。项目以“山海经”为背景，通过退役服务机器人“玄甲”的视角，带孩子们进入一个充满浪漫、体育精神与传统文化魅力的奇幻世界。

## ✨ 核心特性

- **🏃 九大试炼**：互动式山海经试炼卡片，每一项都包含深刻的成长寓意。
- **✨ 山海图鉴**：带拼音标注（如 qǐ zhǒng）的异兽图鉴，支持点击放大查看，助力儿童识字与文化启蒙。
- **🎬 影像记忆**：集成 Bilibili 沉浸式全屏影像播放系统，包含音乐版、语音版及 MV。
- **📖 原创内容**：完整集成《九鼎 runners》九章原创小说及主题曲《跑起来就行》全文。
- **📱 全端适配**：完美支持移动端及平板设备，采用双级响应式布局。
- **🛡️ 知识产权保护**：内置严格的前端防盗用机制（禁止右键、禁止选中、防调试等）。

## 🚀 快速预览

本项目为纯前端静态页面，无需配置后端环境。
1. 下载本仓库。
2. 双击 `index.html` 即可在浏览器中开启奇幻之旅。
